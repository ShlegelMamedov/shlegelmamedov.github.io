# shlegelmamedov.github.io
1. Для чего нужны HTML и CSS?
HTML — язык разметки. Отвечает за структуру и смысловое содержание страницы (заголовки, текст, ссылки, картинки). Это «скелет» сайта.
CSS — язык стилей. Управляет внешним видом и расположением элементов (цвета, шрифты, отступы, сетки). Это «оформление» сайта.
Их разделение упрощает разработку, поддержку и адаптацию страниц.
2. Что такое GitHub Pages и как он работает?
Это бесплатный хостинг от GitHub для публикации статических сайтов (HTML, CSS, JS).
Принцип работы: вы загружаете файлы в репозиторий → в настройках (Settings → Pages) выбираете ветку main → GitHub автоматически собирает и публикует сайт по ссылке username.github.io (или username.github.io/repo). При каждом git push сайт обновляется сам.
3. Использованные теги HTML и свойства CSS:
HTML: <header>, <nav>, <main>, <footer> (семантическая структура); <h1>–<h4>, <p> (контент); <ul>, <li>, <a> (меню и навигация); <link> (подключение стилей).
CSS: display: flex/grid (раскладка); margin/padding (отступы); color, background, font-* (текст и фон); border-radius, box-shadow (визуальное оформление карточек); transition, @media (анимации и адаптивность); position: sticky (фиксация шапки при скролле).
